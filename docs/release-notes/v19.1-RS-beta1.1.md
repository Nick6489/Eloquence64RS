# Eloquence64RS 19.1-RS Beta 1.1

This beta fixes the Enhanced 22 kHz audio mode after Eloquence is reselected
or selected again after switching synthesizers. A newly launched Rust host now
restores the retained audio-quality setting before NVDA opens its matching
audio player, preventing 11 kHz PCM from being played at 22 kHz with an
extremely high pitch.

The Enhanced 22 kHz mode remains otherwise unchanged from Beta 1. Standard
11 kHz audio remains the default, and the production updater remains pinned to
the stable release.
