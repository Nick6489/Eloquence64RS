# Eloquence64RS 19.0-RS-RC2

This release candidate fixes text compatibility and update delivery issues
found while testing RC1.

## Changes since RC1

- Converts capital German sharp S (`ẞ`) to lowercase `ß` before legacy
  code-page conversion so Eloquence speaks it instead of a question mark.
- Moves update checks to the Eloquence64RS release repository.
- Allows release-candidate installations to discover newer prereleases while
  keeping stable installations on stable releases.
- Documents that the secure-screen helper must be copied again after a host
  upgrade.

## Installation

Remove or disable the IBMTTS add-on before installing. Eloquence64RS replaces
Eloquence64 and should not be installed alongside another Eloquence add-on.

After updating, users who need Eloquence on logon or other secure screens must
open **NVDA Settings > Eloquence** and run **Copy Helper to System Config (for
Logon Screen)** again.

## Release-candidate status

Please report speech stalls, synthesizer switching failures, clicks or other
audio artifacts, audio-device issues, and secure-screen failures with an NVDA
debug log and reproduction steps.
